# VideoCompressor Pro - 视频压缩与转换工具

## 功能特性
✅ H.265 → H.264 编码转换
✅ 视频压缩（可调清晰度/质量）
✅ 批量处理
✅ 实时进度显示
✅ 保持原始画面比例
✅ 剩余时间预估

## 使用方法
1. 双击 `VideoCompressorPro.exe` 启动程序
2. 选择转换模式：
   - **编码转换**：H.265 → H.264，保持原始质量
   - **视频压缩**：可调压缩程度和清晰度
3. 添加视频文件（支持 MP4/MKV/AVI/MOV/FLV/WMV/WebM）
4. 选择输出目录（或使用源文件目录）
5. 点击"开始转换"或"开始压缩"

## 压缩参数说明

### 清晰度选项
- **原始**：保持源文件分辨率
- **480p/720p/1080p/2K/4K**：选择目标清晰度

### 压缩程度
- **高质量** (CRF 22)：质量优秀，文件较大
- **中等** (CRF 28)：良好质量，适中大小
- **标准** (CRF 32)：可接受质量，较小文件
- **高压缩** (CRF 40)：一般质量，最小文件

### 编码速度
- **快**：编码快，质量略低
- **中**：平衡（推荐）
- **慢/最慢**：编码慢，质量更好

## 系统要求
- Windows 10 或 Windows 11
- 无需安装 Python 或其他依赖

## 注意事项
- 首次启动会检查 FFmpeg 是否可用
- 如果提示"FFmpeg 不可用"，请将 `ffmpeg.exe` 与程序放在同一目录
- 输出文件自动添加后缀标识（如 `_h264` 或 `_compressed_720p_crf28`）
